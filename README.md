# 📊 ML-Система прогнозирования кадровой нагрузки

Проект разработан в рамках тестового задания на позицию стажера ML-разработчика. Система прогнозирует входящий годовой поток резюме (\`yearly_cv_processed\`) для департаментов компании до 2027 года на основе исторических планов и кадровых метрик.

## 🚀 Архитектура и технологии
* **Стек:** Python, PyTorch, Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, Optuna.
* **Модель:** Кастомная полносвязная нейросеть (MLP) со слоями \`BatchNorm1d\` для стабильности градиентов и \`Dropout\` для регуляризации.
* **Оптимизация:** Автоподбор гиперпараметров (архитектура, lr, weight decay) выполнен через **Optuna** (байесовская оптимизация). 
* **Качество:** Итоговая относительная ошибка модели составила **~9.2% (Val MAPE)** на отложенном валидационном 2024 году.

## 📁 Структура проекта
* \`hr_analys.ipynb\` — Jupyter Notebook с полным циклом (EDA, очистка NaN, масштабирование, Optuna, обучение, бизнес-прогноз до 2027 года).
* \`df_ITs.csv\` — исходный датасет.
* \`best_model/final_top_model.pth\` — сохраненные веса обученной топ-модели.
* \`Dockerfile\` & \`requirements.txt\` — конфигурация для воспроизводимого запуска.

   \`\`\`

2. **Запустите контейнер:**
